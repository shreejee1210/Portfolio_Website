# 🌐 Personal Portfolio Website

## Project Description

This project is a modern and responsive Personal Portfolio Website developed using HTML, CSS, and JavaScript.

The website is designed to showcase my academic background, technical skills, internships, projects, certifications, and contact information in a professional and visually appealing manner. It serves as my digital portfolio and highlights my journey as an Electronics and Communication Engineering student and aspiring developer.

---

## Objective

The main objectives of this project are:

* To create a professional online portfolio
* To showcase skills, projects, and achievements
* To improve frontend web development skills
* To learn responsive web design principles
* To build an attractive and user-friendly interface
